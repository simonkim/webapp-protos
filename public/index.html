FAQ

What mobile phones currently supported?
We currently have an iPhone version in the development phase that is going to be release first. Development of Android and Web Browser versions will be started soon.
I have a WordPress blog, should I switch to Short Stories?
You can keep your WordPress blog. Stories shared to Short Stories can be optionally shared to your WordPress blog.
